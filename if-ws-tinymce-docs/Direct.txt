% Direct 

`SimpleHtmlEditor` and `AdvancedHtmlEditor` formlets provide easy way to create `TinyMCE` editor.
`TinyMCE` editor can be created in similar way to that in `JavaScript` by accessing direct bindings.

Following example shows how to create `TinyMCE` editor using direct bindings:

	#fsharp
	type SampleControl () =
		inherit Web.Control()
	
		[<JavaScript>]
		let Init () =
	
			TinyMCE.Init (
				new TinyMCEConfiguration(
					Theme = "simple",
					Mode = Mode.Textareas
				)
			)
	
		[<JavaScript>]
		override this.Body = 
			TextArea [Id "test_area"] -< [Text "Default  text"]
			|>! OnAfterRender (fun _ ->
				Init()
			)
			:> _
